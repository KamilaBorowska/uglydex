Set

Rayquaza @ Life Orb
Ability: Air Lock
EVs: 252 Atk / 4 Def / 252 Spe
Jolly Nature
- Swords Dance
- Dragon Ascent
- Earthquake
- Extreme Speed

Rayquaza gets a badass 115 Speed, allowing you to outspeed pretty much every Uber Dragon not named Mega Salamence or Dragon Arceus. This also makes Swords Dance a lot more deadly. Dragon Ascent is your main STAB, and it hits like a truck, with only Defense and Special Defense going down. Earthquake hits Steel-types like Dialga and Heatran, plus it hits Primal Groudon a bit harder than Dragon Ascent. Extreme Speed lets you clean teams, but note that Life Orb Rayquaza has a stronger Extreme Speed.

Set

Rayquaza @ Life Orb / Lum Berry
Ability: Air Lock
EVs: 252 Atk / 4 Def / 252 Spe
Naive / Jolly Nature
- Dragon Dance
- Dragon Ascent
- Earthquake
- Draco Meteor / Dragon Claw / Swords Dance

Probably not as good of a sweeper as the Swords Dance set, but this does allow Mega Rayquaza to outspeed some serious stuff such as Arceus, Mega Salamence, Choice Scarf users like Xerneas, etc. Earthquake still 2HKOes Primal Groudon after a boost, which is nice. Draco Meteor lets you smack Dragons such as Palkia without a boost, and also does over 60% to Primal Groudon switch-ins. Dragon Claw is usable if you don't want to go mixed and just wanna smash Palkia and Zekrom's faces in. If you don't care about Dragons, you can run Extreme Speed for some priority in a pinch. A double dance Mega Rayquaza set with Lum Berry is also viable.

Set

Rayquaza @ Life Orb
Ability: Air Lock
EVs: 252 Atk / 4 Def / 252 Spe
Jolly / Adamant Nature
- Dragon Dance
- Dragon Ascent
- Extreme Speed
- Protect / Earthquake / V-create

Dragon Dance set for Doubles. Use with Follow Me/pair with stuff that beats Steel-types. You can run V-create but that's not too conducive to sweeping. Togekiss is an excellent Follow Me partner because Delta Stream eliminates its Rock-, Ice-, and Electric-type weaknesses. Jirachi is nice too.
